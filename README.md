# Seg34Lite.CN
修改自Segment34 MkII表盘([https://github.com/ludw/Segment34mkII](https://github.com/ludw/Segment34mkII))，感谢原作者开源！添加了农历和二十四节气显示，顺便对设置菜单和天气也做了汉化，并精简了很多功能，以适配运行内存比较紧张的老款设备。

## 更新日志
